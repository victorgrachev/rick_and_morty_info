# rick_and_morty_info

---

Для запуска приложения необходимо иметь установленную платформу NodeJS. <br>

# Installation

npm install

# Running the app

#build <br>
npm run build <br>

#development <br>
npm start <br>
После выполнения команды приложение будет доступно по адресу: http://localhost:3000/
